Cortexbias

Field: Neuroeconomics | JEL Codes: D87, D91

Definition

Systematic decision-making deviations arising from the neural architecture and dynamic balance between prefrontal cortex deliberative processes and limbic system emotional responses. Cortexbias manifests when the relative activation levels of these brain regions produce consistent patterns of economic choice that deviate from normative rationality predictions.

Etymology

Compound: 'cortex' + 'bias'.

Usage Example

The endowment effect exhibits cortexbias, with fMRI studies showing amygdala activation driving loss aversion that overrides prefrontal valuation processes.
